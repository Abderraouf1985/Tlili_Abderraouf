Projet 2:

Application Web (Asp.net core) pour
la gestion des articles

C'est un projet de Développement Web en utilisant asp.net core 6 donc j'ai créé des pages Web en utilisant Bootstrap pour avoir un beau design. J'ai ensuite ajouté une page web nommée ‘Items’ pour l’ajout, la mise à jour, la suppression des articles. Toutes les données utilisées pour les opération CRUD dans cette page web seront transféré sur la base de données sur MSSQL. J'ai aussi divisé les articles en 3 catégories : Electric machines, Mobiles, Computers.
